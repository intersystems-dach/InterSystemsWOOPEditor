InterSystems Workshop Organizer and Optimizer Program HELP

This program is designed to help you organize and optimize your InterSystems workshop. It will help you to determine the best order to present the material.

Connect to an instance

Click on the Settings Icon in the top right corner of the screen. Select Advanced and go to the Server tab. Type in the host adress and port number of the instance you want to connect to and click Connect. You can store the connection information by giving the connection a name and click on Save.You can then select the connection from the Connection dropdown menu.

Navigation

  • You can always click on the logo in the top left corner to go back to the home screen.
  • You can click on Log in in the top right corner to log in to account or when you are already logged in, you can click on your name to log out of your account.
  • You can click on the Settings Icon in the top right corner of the screen to open the shortcut settings. From there you can select Advanced to open the advanced settings.

Create a new Chapter

You have to be logged in to create a new chapter!

  1. Go to the home screen.
  2. Click on the + icon.
  3. Fill in the following information:
    • Name - The name of the chapter. This will be displayed in the chapter list.
    • Language - The language of the chapter.
    • Password - You can set a password for the chapter. If you leave this field empty, the chapter will be public.
    • Private - If enabeled the chapter will be private and only accessible by you and the admins.
    • Description - Add a description to the chapter. This will be displayed in the chapter list.
  4. Click on Create to save the chapter.

Edit a chapter

You have to be logged in to edit a chapter!

You can edit a chapter by clicking on the Edit icon in the chapter list. Write the content of your chapters in markdown. You can find a markdown cheat sheet by clicking on the Help button in the editor.

You can add a hint and a result on each page of your chapter. If you leave the hint or result empty, they will not be displayed.

You can add and remove a page by clicking on Add Page and Remove Page. To show a Preview of your chapter, click on Preview. To save your changes, click on Save. You can see if there are unsaved changes by looking at the Save button. If there *Save is displayed, there are unsaved changes. You can turn on autosave by clicking on Autosave. If autosave is turned on, your changes will be saved every 10 seconds. You can edit the metadata of your chapter by clicking on Edit meta data.

Create your own instance

Before you can create your own instance, you need to have a InterSystems IRIS Dataplatform instance running.

  1. Download the InterSystems Workshop Organizer and Optimizer Program from Github.
  2. In the Management Portal, go to System Explorer and select Classes. Click on Import and select the cls folder from the downloaded program.
  3. Open an InterSystems IRIS terminal and go to the namespace where you just imported the classes.
  4. Run the following command: Do ##class(WOOP.Installer).SetUp()
  5. After successful installation, you can access connect to the instance by clicking on the Settings Icon in the top right corner of the screen. Select Advanced and go to the Server tab. Type in the host adress and port number of the instance you want to connect to and click Connect.
  6. You can log in with the following credentials:
    Username: _system
    Password: woop
  7. Then go to the User tab and click on Change Password to change the password.
